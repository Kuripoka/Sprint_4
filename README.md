**Перечень реализованных тестов:**
1) test_add_new_book_add_two_books  
*Добавление двух книг и проверка, что обе появились в коллекции (тест изначально присутствовал в файле после клонирования учебного репозитория)*

2) test_add_new_book_over_forty_one_symbols_name  
*Проверка валидации длины названия книги, не более 40 символов*

3) test_get_book_genre_existing_genre  
*Установка существующего жанра для существующей книги и его проверка*

4) test_set_book_genre_non_existing_genre  
*Попытка установки несуществующего жанра для существующей книги и проверка результата попытки*

5) test_get_books_with_specific_genre_sci_fi  
*Получение списка книг определённого жанра*

6) test_get_books_genre_full_added_books_dictionary  
*Проверка полного словаря после добавления книг и проставления им жанров*

7) test_get_books_for_children_age_restricted_genres_filtered  
*Проверка, что книги с возрастными жанрами («Ужасы», «Детективы») исключаются из детского списка*

8) test_add_book_in_favorites_no_duplicates  
*Добавление книги в избранное без дублей*

9) test_delete_book_from_favorites_book_exists_in_favorites  
*Удаление ранее добавленной в избранное книги из избранного*

10) test_get_list_of_favorites_books_returns_current_state  
*Проверка корректности текущего списка избранных книг после множественного добавления и удаления*
